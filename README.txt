O EventCrafter é uma plataforma concebida para simplificar a gestão e o planeamento de eventos. 
Dirigido a empresas e organizadores e a clientes do ramo, o sistema oferece uma solução integrada que abrange a solução dos problemas que existem na área. 
Com funcionalidades como registo de utilizadores, autenticação, gestão de convidados, pesquisa e reserva de serviços.
O desenvolvimento do EventCrafter envolveu a utilização de diversas tecnologias. Foi utilizada a plataforma Firebase, com o Firestore para armazenamento 
de dados e o Firebase Authentication para a gestão de utilizadores. Foi também utilizado HTML, CSS e JavaScript, recorrendo ao framework Bootstrap para garantir 
um design responsivo e atraente.
